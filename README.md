# [creator]()

## how to use

```sh
. ./env/up.sh app
```

## tools

- [ ] record screen (FFMPEG)

- [ ] split video in images (FFMPEG)

- [ ] write narration(s)

- [ ] generate audio by narration text (ELEVENLABS API)

- [ ] join image and audio (FFMPEG)

- [ ] join video parts (FFMPEG)

- [ ] uploa video (YOUTUBE API)
